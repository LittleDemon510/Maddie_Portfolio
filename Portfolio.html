Capacity Management Tool - PowerApps

I designed and developed a Capacity Management Tool using PowerApps to streamline resource planning for Agile teams.

  • Data Modeling – Snowflake Schema (Dim-Fact Model):Designed the database architecture using a Dimensional Fact Model, where dimension tables store user roles, teams, leaves, calendar data, and PIs, while fact tables track the assignments.

  • User Interface & Role-Based Access Control:
    • Designed all seven screens of the PowerApp, ensuring an intuitive and seamless experience.
    • Implemented automated login using Microsoft Vodafone email addresses, where users are assigned to specific views based on predefined roles in the database.

    • Scrum Master View:
      • Manage and modify Program Increments (PIs) (add, edit, view).
      • View and manage team members, including adding/removing members and changing teams.
      • Access a capacity percentage dashboard to monitor available developer and tester resources for each PI.

    • Developer & Tester View:
      • Submit leave requests via a dedicated dashboard.
      • System automatically updates team capacity based on submitted leaves.

    • Capacity Calculation & Reporting:
      • The tool dynamically calculates team capacity based on submitted leave records and existing team allocations.
      • Provides real-time insights into available resources per PI, helping Scrum Masters make informed decisions.
      • Enables data-driven planning by offering visual capacity reports to track workload distribution and optimize resource utilization.
